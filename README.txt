Software accompaniment to:

Three-way clustering of multi-tissue multi-individual gene expression data using semi-nonnegative tensor decomposition. Annals of Applied Statistics, in press. 

Miaoyan Wang, Jonathan Fischer, and Yun S. Song
University of California, Berkeley

The software folder contains Matlab functions and examples for semi-nonnegative tensor decomposition. 
The supplementary data contain inferred gene modules in the GTEx data analysis.
